Developed an IoT-based gas leakage detection system using MQ2 gas sensor and
flame sensor for real-time safety monitoring.
Integrated ESP8266 Wi-Fi module with Telegram Bot API to send instant alert
notifications during gas leakage or fire detection.
Implemented buzzer alert system for emergency warning and quick response.
Designed a compact and low-power safety solution suitable for homes, laboratories,
and small industries.
Technologies Used: Arduino IDE, ESP8266, MQ2 Sensor, Flame Sensor, Telegram
Bot API, IoT, Embedded
